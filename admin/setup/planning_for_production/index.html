Planning for Production¶

This section provides information to set up LeoFS on the production environment. Planning for production involves ensuring that the hardware and software requirements are met, and the deployment and security consideration are taken into account before installing LeoFS. You can choose to install LeoFS on-premises on supported platforms or virtualization platforms.

Before you begin the actual installation, ensure that you go through the section. For installation instructions on different platforms:

  • On-premises
  • Virtualization platforms:
    • Amazon Web Services and Docker
